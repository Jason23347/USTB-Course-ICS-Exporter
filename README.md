# 北科大课程表 iCalendar 转换脚本

## 本项目用于将[北京科技大学教务微服务](http://jwstu.ustb.edu.cn/)中提供的[网页版课表](http://jwstu.ustb.edu.cn/stu_daily/#/courseSchedule)导出为 iCalendar 文件，便于使用各种日历软件管理课程。

## 使用方法

**方法 1**

添加收藏夹书签

- 名称：微教务课表导出脚本
- 网址：`javascript:void(function(u,s){s=document.body.appendChild(document.createElement('script'));s.src=u;s.charset='UTF-8'}('http://raw.githack.com/NicodeSS/USTB-Course-ICS-Exporter/master/CourseExporter.js'))`
